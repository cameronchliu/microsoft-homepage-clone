# Microsoft Homepage Clone

A responsive clone of Microsoft Taiwan homepage.

## Tech Stack
- HTML5
- CSS3 (Flexbox, Media Queries)
- JavaScript
- Swiper.js

## Features
- Responsive layout for desktop and mobile
- Multi-level navigation menu (PC & mobile)
- Swiper hero banner
- Responsive images using `<picture>`

## What I Learned
- Handling complex layouts with Flexbox
- Building navigation without heavy JS
- Debugging responsive issues across breakpoints
